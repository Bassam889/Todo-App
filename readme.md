# Todo App built with React and SASS

App created for the purpose of storing todos and complete them

## Installation

Install my-project with vite

```bash
  npm create vite
```

Then follow the prompts

Install sass dependencies

```bash
  npm add -D sass
```

# Deployment

To deploy this project run

```bash
  cd my-project
  npm install
  npm run dev
```
